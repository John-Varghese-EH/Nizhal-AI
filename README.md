## ‚ö° Try It Now

Ready to meet your new friend? It takes less than 5 minutes.

### 1. Download & Install
Grab the latest installer for your OS from the [Releases Page](https://github.com/John-Varghese-EH/Nizhal-AI/releases).
*(Windows, macOS, and Linux supported)*

### 2. Developer Setup (Source Code)
If you prefer to dig into the code:

```bash
# Clone the repo
git clone https://github.com/John-Varghese-EH/Nizhal-AI.git

# Install dependencies (Node.js v20+)
npm install

# Start the magic
npm run dev
```

> **Note:** For voice features, ensure a microphone is connected. For local AI, install [Ollama](https://ollama.com).
